# Туризм

Простой одностраничный сайт на чистом HTML, CSS и JavaScript — без фреймворков и сборщиков.

## О проекте

Сайт **Туризм** содержит главную секцию, блок «О проекте», интерактивный раздел «Услуги» и форму обратной связи. Интерфейс адаптирован под мобильные устройства и поддерживает светлую и тёмную тему.

## Возможности

- Адаптивная вёрстка
- Переключение светлой и тёмной темы (с сохранением выбора в `localStorage`)
- Мобильное меню
- Интерактивный блок услуг
- Валидация формы обратной связи в браузере
- Плавная прокрутка по якорным ссылкам

## Структура проекта

```
simple_website/
├── index.html   # Разметка страницы
├── styles.css   # Стили и CSS-переменные для тем
├── script.js    # Интерактивность и логика формы
└── README.md    # Документация проекта
```

## Запуск

1. Клонируйте репозиторий или откройте папку проекта.
2. Откройте файл `index.html` в браузере.

Дополнительная установка зависимостей не требуется.

### Локальный сервер (опционально)

Если нужен локальный сервер, можно использовать расширение **Live Server** в VS Code / Cursor или любой другой статический сервер.

## Технологии

- HTML5
- CSS3 (CSS-переменные, flexbox, media queries)
- JavaScript (Vanilla JS)
- Google Fonts: DM Serif Display, Manrope

## Разделы сайта

| Раздел    | Описание                                      |
|-----------|-----------------------------------------------|
| Главная   | Приветственный блок с кнопками действий       |
